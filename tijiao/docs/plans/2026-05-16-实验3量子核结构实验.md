# 实验3量子核结构实验 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 在远程环境中构造一个小规模量子特征映射实验，验证量子核矩阵是否具有非平凡结构、区分能力和平滑性，并与经典 RBF 核做对比。

**Architecture:** 从 Lorenz96 训练集提取少量代表性局部窗口特征，使用 `4` qubits 的浅层参数化特征映射电路，通过 `Statevector` 计算样本间保真度形成量子核矩阵。实验只分析核矩阵结构，不接入 LETKF 主循环，并同步输出热图、特征值分布、跨时间步对比和文本摘要。

**Tech Stack:** Python 3、qiskit 2.4、numpy、matplotlib、Markdown、远程 Ubuntu 环境

**Step 1: 固定输入与规模**

- 选择早期 / 中期 / 后期 3 个代表时间步。
- 选择小批局部窗口样本，不超过 `16` 个。

**Step 2: 固定评估指标**

- 检查：
  - 核矩阵是否非负
  - 是否不是常数矩阵
  - 是否近似半正定
  - 不同时间步是否有明显变化
  - 与经典 RBF 核相比是否更有结构或至少不更乱
