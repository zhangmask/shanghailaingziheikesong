## 🎯 核心发现

1. **GPU 甜蜜点: 21 qubits** — 低于此值 CPU 更快，24q 时达 4.26x
2. **density_matrix 是 GPU 最大受益者** — 11.74x 加速！
3. **tensor_network 是沐曦适配 bug** — 需沐曦修复
4. **pyscf 安装需要 --network host + 内部源**
5. **VQE/QPE 可通过 StatevectorEstimator 实现** — 绕过兼容问题
