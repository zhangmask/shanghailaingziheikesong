## 三、实验设置

### 3.1 接入原则

- 保留最终赛题兼容的 `result.csv` 输出链路；
- 只替换观测权重的构造方式。

### 3.2 两类混合方案

#### 1. `linear_quantum_mix`

用量子核对固定等权做线性混合：

```math
w = (1 - \lambda) + \lambda K_q
```

含义：

- `lambda = 0` 时退化为固定等权；
- `lambda` 越大，量子核影响越强；
- 这是最温和、最接近基线的量子调制方式。

#### 2. `distance_quantum_mix`

先保留距离局部化，再叠加量子核调制：

```math
w = w_{dist} \cdot ((1 - \lambda) + \lambda K_q)
```

含义：

- 局部距离先验仍然存在；
- 量子核只在局部窗口内部做细粒度调节；
- 更贴近最终赛题可能接受的稳健方案。

```python
def stable_inverse(mat: np.ndarray) -> np.ndarray:
    eye = np.eye(mat.shape[0], dtype=float)
    for jitter in (0.0, 1e-8, 1e-6, 1e-4):
        try:
            return np.linalg.inv(mat + jitter * eye)
        except np.linalg.LinAlgError:
            continue
    return np.linalg.pinv(mat, hermitian=True)
```

- 这说明量子核影响一旦过强，仍会重复 `实验 4` 中的失配问题。

## 六、结果解读

### 6.1 混合比完全替代更合理，但当前仍未形成正增益

这轮实验最重要的信号是：

> 量子核做温和混合，明显比“直接替代对角权重”更稳，但当前版本仍没有真正超过经典 `fixed` 基线。

也就是说：

- `实验 4` 失败说明“直接替代”不行；
- `实验 5` 说明“温和混合”方向更合理；
- 但“方向更合理”不等于“已经形成最终有效的方法”。

### 6.2 线性混合优于距离混合

从结果上看：

- `linear_quantum_mix, lambda = 0.1` 是所有混合方案中最好的；
- `distance_quantum_mix` 在所有测试点上都没有超过它；
- 说明当前量子核更适合做对固定等权的小幅调制，而不是再叠加额外距离因子。

这也意味着当前距离先验与量子核的乘法耦合，并没有形成额外增益。

### 6.3 当前量子核更像“弱调制信号”，还不是“主导权重信号”

从训练集和测试集一起看，可以得到更细的判断：

- 小 `lambda` 时，量子核能提供一点点可利用的结构信息；
- 但这点信息还不足以在泛化集上超过经典强基线；
- 一旦把量子核权重放大，它就会迅速把更新方向拉偏。

因此当前更准确的表述不是“量子核无用”，而是：

> 当前这版量子核只能作为弱调制信号，暂时还不能成为最终赛题方案中的主导观测权重。

## 七、当前结论

本次实验最重要的一句话是：

> 量子核混合权重方案比“直接量子对角加权”稳定得多，但在当前特征映射、当前混合方式和当前参数范围下，仍没有任何方案超过 `fixed` 基线；因此，量子核目前还没有对最终赛题方法形成明确正贡献。

换句话说：

- `实验 3` 证明了量子核有结构；
- `实验 4` 否定了量子核直接替代对角权重；
- `实验 5` 进一步说明温和混合是更合理的接法，但当前仍不足以替代经典基线。

当前最强可交付方案仍然是：

- `fixed`

## 九、后续建议

如果继续沿量子核方向推进，更合理的下一步不是继续加大 `lambda`，而是：

1. 重新设计量子特征映射  
   让量子核更贴近“观测可信度”而不只是“窗口相似度”。

2. 让量子核先作用于表示层或样本分组  
   再间接影响同化，而不是继续直接进对角权重。

3. 只围绕极小强度做更细扫描  
   例如在 `0.02` 到 `0.15` 之间做更细粒度搜索，判断是否存在真正优于 `fixed` 的窄窗口。

4. 保持经典 `fixed` 作为当前主线基线  
   量子方案暂时不要替代主提交链路。
