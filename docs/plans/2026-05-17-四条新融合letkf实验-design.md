# 四条新融合 LETKF 实验 Design

## 目标

在 `exp32-36` 已经覆盖 `rho + R`、`rho` 自适应、`R` 自适应、`rho + R + B`、`rho + B` 之后，继续并行开 4 条新的最小弱融合线，回答两个问题：

- 量子结构信号除了 `rho / R / B` 外，是否还能在 `J` 与 `inflation` 上作为弱控制发挥作用
- 已有次级正信号之间，是否存在比当前主线更轻但可互补的组合

这轮不追求一次性做深做广，而是：

> 用 4 条“最小弱融合版”脚本，把新的可能方向先快速验证出来。

## 四条实验

### exp37: rho + J

- 脚本：`exp37_rho_J_weak_fusion_letkf.py`
- 角色分工：
  - `rho` 负责局部观测参与
  - `J` 只做末端弱几何修正
- 目标：
  - 看 `J` 的几何信息在 `rho` 主线下是否仍有正信号

### exp38: rho + inflation

- 脚本：`exp38_rho_inflation_weak_fusion_letkf.py`
- 角色分工：
  - `rho` 主导局地化
  - 量子信号只弱调制局部 inflation
- 目标：
  - 看泛化误差能否优于单独 `rho`

### exp39: rho + adaptive_B

- 脚本：`exp39_rho_adaptive_B_letkf.py`
- 角色分工：
  - `rho` 主导局部观测参与
  - `B` 不再固定 `lambda`，而是很保守的局部自适应
- 目标：
  - 看 `rho + B` 能否在 `exp36` 基础上继续抬升

### exp40: R + B

- 脚本：`exp40_R_B_weak_fusion_letkf.py`
- 角色分工：
  - `R` 负责结构化观测误差
  - `B` 做末端弱修正
- 目标：
  - 看两个次级正信号是否在不依赖 `rho` 时仍能互补

## 统一原则

- 4 条都做“最小弱融合版”
- 4 条都保留 `fixed` 与 `corr` 对照
- 首轮都只做固定小参数
- 不叠第四个对象
- 除 `exp39` 外，其余不做自适应
- 全部静默挂远端

## 参数策略

### exp37

- `lam_rho = [0.02, 0.03]`
- `lam_j = [0.003, 0.005]`

### exp38

- `lam_rho = [0.02, 0.03]`
- `lam_infl = [0.01, 0.02]`

### exp39

- `lam_rho = [0.02, 0.03]`
- `lambda_b_i` 由局部信号保守裁剪到 `[0.002, 0.008]`

### exp40

- `lam_r = [0.02, 0.05]`
- `lam_b = [0.003, 0.005]`

## 融合顺序

### exp37

1. `rho` 形成观测参与权重
2. 经典 LETKF 求 `w_mean / w_pert`
3. `J` 做末端弱几何修正

### exp38

1. `rho` 形成观测参与权重
2. 量子信号弱调制局部 inflation
3. 用调制后的 inflation 做经典分析

### exp39

1. `rho` 形成观测参与权重
2. 局部信号给出 `lambda_b_i`
3. `adaptive_B` 做末端弱修正

### exp40

1. 结构化 `R`
2. 经典 LETKF 求 `w_mean / w_pert`
3. `B` 做末端弱修正

## 验收标准

- `exp37` 至少优于单独 `J` 线的已知表现
- `exp38` 至少不明显差于 `exp13`
- `exp39` 至少优于 `exp36` 中一部分组合
- `exp40` 至少优于 `fixed`，并尽量接近 `exp10 / exp17`

## 止损条件

- 若某条线全部组合显著差于 `fixed`，则该线判为当前无效
- 若某条线训练改善但测试普遍退化，则判为过拟合式接入
- 若出现 `NaN` 或病态矩阵，优先收缩参数，不扩大扫描

## 一句话结论

这轮设计不是继续沿单一主线细抠，  
而是：

> 以“最小弱融合版”为统一模板，把 `rho + J`、`rho + inflation`、`rho + adaptive_B`、`R + B` 四条新方向一次性挂起来，快速筛出下一条真正值得深化的融合主线。
